r=== Cashback Tracker Pro ===
Contributors: keywordrush.com

Cashback tracker.

== Installation ==

**Requirements**
* PHP 5.6+ (note, Wordpress without plugin needs PHP 5.2.4+).
* Wordpress 4.9.8+.

This section describes how to install the plugin and get it working
1. Upload `cashback-tracker` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Configure plugin settings
4. [User Guide](https://www.keywordrush.com/docs/cashback-tracker/)
